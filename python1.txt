LIST:
A list in python is an ordered collection of multiple elements.
-It is mutable
-it is indexed
-can store different data Types
-starting from index [0]

examples:
my_list = [10,20,30,40]
print(my_list)

Access list items:
numbers = [10, 20, 30, 40, 50]
print(numbers[0])     # 10
print(numbers[-1])    # 50
print(numbers[1:4])   # [20, 30, 40]

List methods:  .append()
               .extend()
               .insert()
               .remove()
               .pop()
               .sort()

append(): Add item at end
example:
a = [1, 2, 3]
a.append(4)
print(a)

extend(): Add multiple items
example:
a = [1, 2, 3]
b = [4, 5]
a.extend(b)
print(a)

insert(): Add item at specific index
example:
a = [1, 2, 3]
a.insert(4, 5)
print(a)

remove(): Remove specific value
example:
a = [10, 20, 30]
a.remove(30)
print(a)

pop(): Remove item by index
example:
a = ["A", "B", "C"]
a.pop(2)
print(a)

sort(): Sort the list
example:
a = [30, 10, 20]
a.sort()
print(a)

reverse(): Reverse list
example:
a = [1, 2, 3]
a.reverse()
print(a)


Coversion of String to list:
using split()
example:
text = "apple banana cherry"
result = text.split()
print(result)



DICTIONARY: A dictionary in python is unordered, mutable collection that stores data in key-value pairs.
-keys are unique
-any data Types
-use {}

example:
person = {
    "name": "Pruthvi",
    "age": 23,
    "city": "Bengaluru"
}
print(person)

Dictionary methods: keys()
                    values()
                    items()
                    update()
                    clear()

Access dictionary values:
using key:
print(person["name"])

using get():
print(person.get("age"))


Adding and Updating values:
add new key value pair
example:
person["email"] = "pruthvi@gmail.com"

Update existing values:
person["city"] = "Tumakuru"


Remove items:
remove by key:
person.pop("age")

removes last item:
person.popitem()

delete key value:
del person

clear: Del entire dictionary
example:
person = {
    "name": "Pruthvi",
    "age": 23,
    "city": "Bengaluru"
}
person.clear()
print(person) #output: {}

-faster than list

-allows duplicate values

example:

Tuple methods: -count()

example:

example:

SETS: A set in python is an unordered, unindexed, mutable collection of unique values.
-no duplicate values, removes automatically
-unordered
-using {}

example:
my_set = {1, 2, 3}
print(my_set)

empty set:
s = {}

Set operations: -union() or |
                -intersection() or &
                -difference() or -

union(): adds all unique items from both SETS
example:
a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)
or
print(a.union(b))

intersection(): common items
example:
a = {1, 2, 3}
b = {3, 4, 5}
print(a & b)
or
print(a.intersecion(b))

difference: items in A not in b
a = {1, 2, 3}
b = {3, 4, 5}
print(a - b)
or
print(a.differnec(b))
