LIST:
A list in python is an ordered collection of multiple elements.
It is mutable
It is indexed
can store different data Types
starting from index [0]

examples:
my_list = [10,20,30,40]
print(my_list)

Access list items:
numbers = [10, 20, 30, 40, 50]
print(numbers[0])     # 10
print(numbers[-1])    # 50
print(numbers[1:4])   # [20, 30, 40]
